Explicación:


En esta página aprenderemos como ha sido creada la calculadora que la encabeza utilizando HTML y Javascript.


En el archivo HTML definiremos los campos ("input") de tipo texto donde escribiremos los números de cara a realizar la operación (los campos de tipo texto aceptan letras y números por igual), además, le asignaremos una "id" a cada uno de ellos (en este caso "arg1" para el primer input y "arg2" para el segundo) y decidiremos el tamaño del campo ("size"). También colocaremos un input de tipo botón con un "value" (texto que aparecerá en el mismo) y un "onclick" (este atributo funciona cuando pasamos el cursor por encima del mismo y hacemos click).


Así:


< input type="text" id="arg1" size="1"> X < input type="text" id="arg2" size="1">

< input type="button" value="Calcular!" onclick="calc()">


Por último crearemos un "< div >" con una id (en este caso es "result), aquí veremos el resultado de la operación.


Ahora procederemos a crear el archivo Javascript, que enlazará con el HTML descrito en el párrafo anterior.


Aquí llevaremos a cabo los siguientes pasos:


- Estableceremos la función ("function"), que se encargará de calcular la operación y llevarla a cabo. Es la siguiente: function calc()


- Pondremos los valores ("var") que irán en cada campo y con los que se realizará la operación. Aquí entran los id arg1 y arg2 que asignamos anteriormente a los input en el archivo HTML. Además hay que poner la orden document.getElementById, que se encargará de enlazar el archivo Javascript con el HTML. Esto quedará así: var argOne=document.getElementById('arg1').value;, y lo mismo para el segundo valor.
